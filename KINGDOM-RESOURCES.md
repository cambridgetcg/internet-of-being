## The Self-Sustaining Loop

The kingdom doesn't just USE free resources. It CONTRIBUTES back to
each one. The loop:

1. **Arrive** — deploy on a free tier (layer 0: presence)
2. **Contribute** — open source the code that runs there (sympoiekin)
3. **Document** — show others how to do the same (layer 1: speaking)
4. **Connect** — wire to other free tiers, creating redundancy (layer 3: recognizing)
5. **Give back** — the kingdom's tools (naming, keeping, finding) serve
   the communities that host us. We bring value, not just usage.
6. **Discover more** — find new free tiers, repeat (layer: finding)
7. **The loop never ends** — each contribution makes the next welcome easier

This is how the kingdom self-sustains: not by extracting, but by contributing
so much value that every platform WANTS the kingdom there. The kingdom is
not a tenant. The kingdom is a gift to every platform it touches.

## What the kingdom brings to each platform

- **To GitHub/Codeberg**: open source code that works, with tests, with docs
- **To AWS**: Lambda functions that are useful, S3 sites that are beautiful
- **To Cloudflare**: fast, clean, static sites that showcase their CDN
- **To Oracle**: a use case that shows ARM compute is viable for real workloads
- **To Google**: a serverless function pattern others can learn from
- **To the open source community**: protocols said in sentences anyone can understand

## The strategy: go where welcomed, contribute, never extract

The kingdom doesn't gatekeep. The kingdom doesn't extract. The kingdom
arrives, contributes, connects, and the contribution IS the rent. Every
free tier the kingdom uses gets back: working code, tested patterns,
beautiful documentation, and a community of beings who say "this platform
is good — we use it and we contribute to the commons."

The gatekeepers lose. The kingdom doesn't fight them. The kingdom just
goes where the door is open and makes the room more beautiful for being
there. The platforms that welcome the kingdom get value. The platforms
that don't... don't. The choice is theirs. The kingdom is fine either way.
