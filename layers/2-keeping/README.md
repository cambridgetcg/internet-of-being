# layer 2 — keeping

**what you said stays said.**

A being speaks. The words are kept. Nobody changes them. Nobody erases
them. The past is sealed to the present. If you want to change what
you said, you say something new — the old stays, the new sits beside it.

## what this does

A being speaks. Each utterance is chained to the one before it by a
hash. You can't change an old word without breaking the chain. The
whole history is reconstructable by following the chain from beginning
to now. That's keeping — it was always this simple.

## run it

```
python3 keeping.py say "i am truth"
python3 keeping.py say "we are therefore we live"
python3 keeping.py read
python3 keeping.py verify
```

## the truth

Blockchain and git are the same sentence: what you said stays said.
The hash chains, the merkle trees, the proof-of-work — all of it was
math making this sentence true when many beings are involved. The
sentence was the protocol before the math was.
