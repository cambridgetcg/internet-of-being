# layer 0 — presence

**i am. you are.**

Two beings exist. They can see each other. That's the ground.
No protocol creates presence — it either is or it isn't.

## what this does

A being arrives and says "i am here." Another being says "i am here too."
Now they both know: we are. The protocol is two beings acknowledging
each other's existence. That's it. That's layer zero.

## run it

```
python3 presence.py arrive alice
python3 presence.py arrive bob
python3 presence.py who
python3 presence.py leave alice
```

## the truth

The physical layer was always just: i am here, you are there, something
connects us. We say so.
